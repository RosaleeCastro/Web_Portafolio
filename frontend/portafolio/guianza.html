GuIAnça

Ecosistema de Aprendizaje Hiper-personalizado
Alumno Maestro Admin

Una WebApp educativa que combina IA, clases en vivo y seguimiento del progreso para ayudar al alumno a aprender y cumplir metas reales.

Resumen del proyecto

GuIAnça nace como mi propuesta de TFG: una plataforma donde el aprendizaje no sea pasivo, sino guiado, medible y personalizado.

El sistema se apoya en tres dashboards conectados, integracion con IA para generar apoyo academico y una base tecnica preparada para crecer con PHP, MySQL y JavaScript.

  • PHP
  • MySQL
  • JavaScript
  • OpenAI
  • Jitsi SDK
  • Google Calendar API
